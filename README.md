# Master-Software-Design_assignment01
Hello, My name is Abir. This is my Master Software Design course Assignment01. Topic is : A Software System for Food Ordering and Delivery
----------------------------------------------------------------------------------------------------------------------------------------------The Netfood project aims to develop a delivery-oriented order management system that allows users to order from multiple restaurants simultaneously and helps the work of the delivery personnel in tracking the orders.
There are many applications for food ordering from local restaurants in particular cities. Some systems support group orders too, but these are usually restaurant-oriented: an order can only contain items requested from a single restaurant. This model works well where the delivery can be solved separately by the restaurants, but there are settlements as like small towns where multiple restaurants are working together with the same delivery company.
